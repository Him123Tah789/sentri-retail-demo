# Privacy Policy

## Data Collection

Sentri Retail Demo collects and processes the following types of data:

### User Data
- Account information (username, email)
- Authentication credentials (hashed passwords)
- User preferences and settings
- Activity logs and usage patterns

### Security Data
- Risk assessment results
- Scan logs and reports
- Security incident data
- System performance metrics

### Demo Data
- Sample retail data for demonstration purposes
- Generated security scenarios
- Test user interactions
- Synthetic analytics data

## Data Usage

We use collected data to:
- Provide security monitoring and risk assessment
- Generate insights and analytics
- Improve system performance
- Demonstrate product capabilities
- Ensure system security and integrity

## Data Storage

- Data is stored securely using industry-standard encryption
- Local storage uses SQLite database
- Environment variables protect sensitive configurations
- Regular backups and data integrity checks

## Data Sharing

This demo application:
- Does not share data with third parties
- Processes data locally by default
- May integrate with external APIs for enhanced functionality
- Maintains data isolation between demo sessions

## User Rights

Users have the right to:
- Access their personal data
- Request data correction or deletion
- Export their data
- Withdraw consent for data processing
- Receive transparency about data usage

## Data Retention

- Demo data is retained for the duration of the demo session
- User accounts can be deleted upon request
- System logs are retained for security and debugging purposes
- Backup data follows the same retention policies

## Security Measures

- Data encryption in transit and at rest
- Access controls and authentication
- Regular security audits and updates
- Compliance with data protection regulations

## Contact

For privacy-related questions or requests, contact: privacy@example.com

## Updates

This privacy policy may be updated to reflect changes in functionality or legal requirements. Users will be notified of significant changes.
